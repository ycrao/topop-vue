topop-vue
---------


### usage

```bash
# requirements: node >= 16
# skeleton see: https://cn.vuejs.org/guide/quick-start.html#creating-a-vue-application
npm init vue@latest
# ...

# install deps
cd topop-vue
npm install -g pnpm
pnpm i

# DX
pnpm dev

# unit test
pnpm test:unit
# e2e test
pnpm test:e2e:dev
```
